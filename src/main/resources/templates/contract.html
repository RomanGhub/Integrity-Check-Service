Контракт Програми

Ця сторінка описує контракт програми з користувацькою програмою. Користувацька програма має відповідати кільком критеріям:

// Зразок Java-програми

public class Algorithm implements Serializable {
    public static void main(String[] args) {
        System.out.println("Привіт, світ!");
    }

    public String check(InputStream inputStream) { // байти[]
                try {
            MessageDigest md = MessageDigest.getInstance("SHA-256");
            byte[] buffer = new byte[8192];
            int bytesRead;
            while ((bytesRead = inputStream.read(buffer)) != -1) {
                md.update(buffer, 0, bytesRead);
            }
            byte[] digest = md.digest();
            return bytesToHex(digest);
        } catch (Exception e) {
            e.printStackTrace();
            return null;

        }
    }

    private static String bytesToHex(byte[] bytes) { // Користувацька програма може містити будь-які методи
        StringBuilder result = new StringBuilder();
        for (byte b : bytes) {
            result.append(String.format("%02X", b));
        }
        return result.toString();
    }

Алгоритм, який імплементує ваш метод, має бути детерміністичним. Сигнатура обов'язкового методу check() має бути такою:

  • public String check(InputStream inputStream)

Сигнатура головного класу має бути такою:

  • public class Algorithm implements Serializable
